# agentic-video-summarizer-phidata

### Description

An agentic video summarizer using the PhiData framework. A lightweight app to upload a short video, auto-summarize it, and ask follow-up questions grounded in the videoâ€™s content. Built with Streamlit UI and a PhiData agent using Google Gemini; optionally enables web search for out-of-scope questions.

### Features

Upload a short clip (.mp4, .mov, etc.)

Instant, concise summary

Q&A based only on the video

Optional DuckDuckGo web lookup

### How It Works

Streamlit handles upload and display

PhiData agent (Gemini) analyzes the video

Answers are grounded in the video; uses web search only if enabled
